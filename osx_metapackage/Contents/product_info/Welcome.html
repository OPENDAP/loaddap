The Matlab OPeNDAP Data Tool

The OPeNDAP loaddap Data Tool provides a way for you to read data from many different Internet servers and load those data directly into Matlab without any laborious format translation.

To use this package you will need Matlab 2009a or later.

The following screens will guide you through the installation process, which should be automatic. If you have any questions regarding this software, please contact the authors via support@opendap.org.

Note that this package will install libdap in /usr/lib and it will install the matlab client (loaddap) in /usr/local/bin. You will need to add the directory /usr/local/bin you your matlab path using the addpath command.
